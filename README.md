# Postman-API-codes
Test APIs to potentially host on postman

To run this app, please follow these steps:
1. enter the following command into the terminal to enter the directory: cd fastapi
2. uvicorn main:app --reload
3. Copy and paste the given ip into your web browser (preferably chrome)
4. Include '/docs' at the end of the ip address to initialize a Swagger UI
5. Send get, post, put, or delete requests into the browser
